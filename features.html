Features

Translate any on-screen text in place. Drop a resizable floating window (“번역창”) over what you want to read. It never steals focus or blocks clicks—your apps keep working.

Highlights

  • 🪟 Non-intrusive overlay — no intercepted keystrokes or mouse clicks.
  • 🧠 OCR → AI translation pipeline (Windows OCR/Tesseract + DeepL/OpenAI).
  • ⚡ Fast & natural output with latency-friendly processing.
  • 🔒 Privacy-first — minimal data sent, no logging by default.

How it works

  1. Place the window over the text area.
  2. Capture & OCR extracts clean text (English, Japanese, Korean, etc.).
  3. Translate via DeepL/OpenAI and render inside the same window.

Tech & Privacy

  • Desktop: PyQt overlay, screen capture (MSS), adjustable window.
  • OCR: Windows OCR or Tesseract (tunable per language/scripts).
  • Translation: DeepL/OpenAI (selectable backend).
  • Privacy: Sends only the clipped region for translation if cloud is used; no keystroke capture; no logs unless user enables.
